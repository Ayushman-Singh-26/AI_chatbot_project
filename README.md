# AI_chatbot_project
# 🤖 AI Voice Assistant Chatbot

An AI-powered voice assistant built with Python that listens to voice commands and performs tasks like opening websites, playing music, fetching news, and generating AI responses.

---

## 🚀 Features

- 🎙️ Voice recognition using microphone input
- 🔊 Text-to-speech responses
- 🌐 Open popular websites via voice command
- 🎵 Play music from a custom library
- 📰 Fetch latest news headlines
- 🧠 AI-generated responses using Hugging Face API
- ⚡ Wake word detection ("computer")

---

## 🛠️ Tech Stack

- Python
- speech_recognition
- pyttsx3
- requests
- Hugging Face Inference API
- NewsAPI
